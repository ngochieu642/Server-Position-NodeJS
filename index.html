Autonomous Agri-vehicle Control System

Description Details Files Contributors

Description

Having robots on farms will help negate the need for pesticides, chemicals and destruction of soil structure. The machine need to be fully autonomous and features in the control system should include:

  1. High accuracy, error correcting GPS/GNSS with RTK
  2. Micro processors for controlling electric motors, electric valves and enabling parallel processing. Linked via SPI
  3. Cellular 2G/3G/4G data comms
  4. Object Recognition and positioning for distinguishing weed from soil
  5. Application that points out the robot's position
At present, the system will concentrate on one simple task: weeb prevention

Details

An agricultural robot has different requirements.

  • Super accurate GPS/GNSS-not just one unit, but two, enabling error correction between the two-one is static(base) and the other roving.
  • A mean of communications for the robot to receive signal from base, as well as to send useful information back to Web Server
  • Ai based object recognition that is fast has acceptable accuracy

Files

  • STM32F407VG files: The processing unit of RTK algorithm, communicates with Android Phone and Raspberry
  • STM32F1C308 files: I/O Controller(valves and motors)
  • Raspberry Pi 3 B+ files: Object Recognition CNN
  • Jetson Nano: Can be used as a stronger version of Raspberry
  • Android Phone On board files: Receives GPS/GNSS messages from base
  • Base files: Transmit GPS/GNSS messages from base to rover
